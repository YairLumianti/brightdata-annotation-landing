# Bright Data Annotation — Landing Page

A modern, responsive landing page for **Bright Data Annotation**, an AI/ML data annotation product. Built with pure HTML, CSS, and vanilla JavaScript — no build step required.

![Bright Data Annotation Preview](assets/preview.png)

## ✨ Features

- 🎨 Modern, clean SaaS design
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Zero dependencies, fast load
- 🌈 Custom gradient hero & CTA sections
- 🧩 Modular sections: Hero, Features, Stats, Pricing, Footer
- 🎬 Smooth scroll + fade-in animations
- ♿ Accessible & semantic HTML
